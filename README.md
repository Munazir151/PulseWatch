# PulseWatch

PulseWatch is a cross-platform pharmacovigilance platform for tracking drug safety signals, reviewing alerts, and generating reports from a web dashboard and a mobile app.

It is built to help teams collect public signal data, analyze it with NLP, and turn it into actionable safety insights.

## Key Capabilities

- Public signal ingestion from social and community sources
- Biomedical NLP for drug, symptom, and adverse event detection
- Real-time dashboards for alerts, trends, and regional severity views
- CDSCO-style report export with evidence links and audit trail
- Privacy controls for masking sensitive patient details

## Repository Layout

- `apps/web` - React Router web app for the dashboard and public site
- `apps/mobile` - Expo / React Native app for the mobile experience
